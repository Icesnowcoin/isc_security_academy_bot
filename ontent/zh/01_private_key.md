# 私钥管理：你的资产唯一防线

私钥（Private Key）是区块链资产的唯一控制权凭证。任何人获取你的私钥，即等同于拥有你钱包内的全部资产。

核心原则：
1. 永不截图、永不拍照、永不复制到剪贴板同步云端。
2. 使用物理介质记录：不锈钢助记词板、防水防火纸张，存放于保险箱或分散保管。
3. 冷钱包（离线设备）存储大额资产，热钱包仅留日常小额。
4. 定期审查设备是否被植入键盘记录器或恶意软件。
