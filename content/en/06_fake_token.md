# Fake Token Identification: Don't Be Fooled by Names

In the decentralized world, anyone can create a token with the same name as a famous project. Attackers exploit this to trick users into buying worthless fake tokens on DEXs.

### How to Verify a Token
The most important factor is the **Contract Address**, which is a unique identifier. While names can be duplicated, addresses cannot. Always verify the address via official project channels or CoinMarketCap. Additionally, check the **Liquidity**—fake tokens often have very low liquidity or are "honeypots" where you can buy but not sell. Finally, examine the **Holder Distribution** on BscScan; if a few addresses hold 99% of the supply without locking, it is a massive red flag.
